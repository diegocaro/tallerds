Series¶

  • Una serie es una coleccin de valores.
In [2]:
pd.Series(['Santiago', 'Concepcin', 'Valparaso'])
Out[2]:
0      Santiago
1    Concepcin
2    Valparaso
dtype: object
  • Estn compuestos por un ndice y valores
In [3]:
s = pd.Series([0.25, 0.5, 0.75, 1.0])
s
Out[3]:
0    0.25
1    0.50
2    0.75
3    1.00
dtype: float64
In [4]:
s.index
Out[4]:
RangeIndex(start=0, stop=4, step=1)
In [5]:
s.values
Out[5]:
array([ 0.25,  0.5 ,  0.75,  1.  ])

Operaciones sobre Series¶

  • Operaciones aritmticas a cada elemento de la serie (como un vector de matlab)
    • Suma, resta, divisin, etc...
    • Ms info en https://pandas.pydata.org/pandas-docs/stable/dsintro.html
In [6]:
s + 50 # suma 50 a cada valor de la serie
Out[6]:
0    50.25
1    50.50
2    50.75
3    51.00
dtype: float64
In [7]:
s * 3.1416
Out[7]:
0    0.7854
1    1.5708
2    2.3562
3    3.1416
dtype: float64
  • Tambin podemos realizar operaciones entre dos series (deben ser del mismo tamao)
In [8]:
h = pd.Series([-100, -99, -98, -97])
h
Out[8]:
0   -100
1    -99
2    -98
3    -97
dtype: int64
In [9]:
s + h
Out[9]:
0   -99.75
1   -98.50
2   -97.25
3   -96.00
dtype: float64
  • Y tambin se pueden realizar operaciones sobre todos los valores
In [10]:
s.sum()
Out[10]:
2.5
In [11]:
s.mean()
Out[11]:
0.625
In [12]:
s.std()
Out[12]:
0.3227486121839514

Consultando series¶

  • Podemos recuperar elementos de la serie indicando su posicin (recuperar el i-simo elemento)
In [13]:
s = pd.Series([0.25, 0.5, 0.75, 1.0])
In [14]:
s.loc[2] #recupera el segundo elemento de s
Out[14]:
0.75
In [15]:
s.loc[2:3] #recupera un rango de elementos
Out[15]:
2    0.75
3    1.00
dtype: float64
  • Y tambin podemos seleccionar por alguna condicin sobre los valores
In [16]:
s[s > 0.4] # puede usar cualquier operador de comparacin
Out[16]:
1    0.50
2    0.75
3    1.00
dtype: float64
  • El operador de comparacin devuelve una serie de True/False
    • Todas las posiciones que son True se seleccionan
In [17]:
s > 0.4
Out[17]:
0    False
1     True
2     True
3     True
dtype: bool

Dataframe¶

  • Es una tabla (si, como las de Excel), donde cada columna es una Serie.
In [18]:
area = pd.Series({'California': 423967, 'Texas': 695662,
                  'New York': 141297, 'Florida': 170312,
                  'Illinois': 149995})
pop = pd.Series({'California': 38332521, 'Texas': 26448193,
                 'New York': 19651127, 'Florida': 19552860,
                 'Illinois': 12882135})
df = pd.DataFrame({'area':area, 'pop':pop})
df

Las condiciones se pueden mezclar con el operador or, and y not.

Operacin Python (if) Pandas
Disyuncin or |
Conjuncin and &
Negacin not ~
